Basic Computer OS

OS (Operating System) is a software that acts as an interface between computer and the user, which is the applications like Browsers, This PC, and etc. A primary goal of an Operating System is to make all the applications and hardware attached to the computer system accessible to all the users. Some popular Operating Systems include Linux, Windows, VMS, OS/400, AIX, z/OS, Mac etc.

Types of OS

Basically, there are 6 types of OS

  • Windows
  • macOS
  • Linux
  • chromeOS
  • Android
  • iOS

    • Windows

    Windows is basically developed by Microsoft. Windows is used for personal and professional computing. It has higher graphical user interface (GUI) and more spaces for wide storage of application than most of other OS. 

    Examples: Windows 10, Windows 11.

    macOS

    macOS is developed by Apple for their Mac computers. It has a smooth design as the same as most Apple devices. 

    Example: macOS Mojave, macOS Sierra

    Linux

    Linux is one of the most widely used operating system of all time. Much known for its highly secured system and low price and license fees. 

    Example: Bootloader, Kernel

    chromeOS

    chromeOS is developed by google and it's mostly focused on cloud-based applications. It started becoming one of the most used popular device in around 2020-2021, following with expensive cost due to its lightweight and durability.

    Android

    Android is a touchscreen operating system which is mostly focused on smartphones. It can be known as second-third most used operation system on mobile around the world. 

    Example: Samsaung, Hawaii

    iOS

    iOS is the most developed operation system from Apple. It's also one of the most world wide used OS which is at the same popularity level as Androids. 

    Example: iPhone, iPad
