Basic PowerPoint Tasks

PowerPoint is a presentation Microsoft software that is used to create slideshows, and multimedia. We're going to show the very basics of PowerPoint.

Creating Slides:

Go to Home or Insert section. You'll see "New Slides". Click it and you'll get a new slide. Click the downward arrow for more types of slides you want to create. You can also get a new slide by right clicking the slides view part and clicking the create new slide button.

Inserting Elements:

To insert shapes, go to Home or Insert section. You'll see "shapes" part. Click it and you will see variety of shapes and choose one you want to create. If you want to add Media (Image, Video, Audio), you can go to Insert section and you will see the buttons to add media.

Presenting and Recording:

If you want to start the presentation in PowerPoint, click the Slide Show button with a projector slide logo at the bottom left side of the whole application. If you want to record, go to Record section and you will see variety of options to Record. You can record with both camera and audio.

Exporting:

If you want to export your PowerPoint file to another type of file, go to File section and go to Export. and and you can choose which type of file you want to export your PowerPoint file into.
