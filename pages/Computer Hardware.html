Basic Computer Hardware

Computer Hardware is input/output devices connected to the computer. Basically, it's any part of the computer you can physically touch.
We'll devide this into 3 parts. 

1. Essential Hardwares
2. Input Devices
3. Output Devices

Essential Hardwares


Central Proceessing Unit (CPU): Also known as "the brain of a computer". It performs commmands and executions of the computer.


Memory (RAM): It's a storage for data but only for the datas which are actively used.


Storage Device: There're 2 storage devices, HDD(Hard Disk Drive) and SSD(Solid State Drive). 

HDD: Big storage, Slow, Cheap.
SSD: Smaller storage, Fast, A bit more expensive.


MotherBoard It's the central part of a computer where all the ciruits and components are connected.

Input Devices


Mouse: A device that is used to control the computer with a display called "cursor" on the screen.


Keyboard: The device used to import commands and letters into the computer hardware.

Output Devices


Speaker/Headphone: Output devices that are used to hear the sounds made by computer device.


Moniter: It's the output device mostly used on PC. It's used to display the screen of a computer.
